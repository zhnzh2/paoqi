# 炮棋 (Paoqi)

一个基于 Python 实现的策略棋类游戏，目前为命令行版本，支持完整对局流程、炮管机制与棋谱记录。

---

## 🎮 项目简介

炮棋是一种围绕“炮管生成与发射”机制设计的策略棋类游戏。  
玩家通过落子、形成炮管、发射与吃子，不断扩大优势，最终以棋子数量决定胜负。

---

## ✨ 当前功能

- ✅ 9×9 棋盘系统
- ✅ 棋子等级（1–5级）
- ✅ 炮管自动识别与发射机制
- ✅ 吃子判定系统
- ✅ 回合自动结算（打炮 → 吃子循环）
- ✅ 多炮同时发射支持
- ✅ 棋谱记录（history）
- ✅ 撤销操作（undo）

```bash
python main.py
```

## 🧩 核心规则（简要）

### 落子

- 只能在己方棋子**正交相邻位置**落子  
- 或对己方低级棋子进行升级  

---

### 炮管

- 连续 ≥3 个同等级棋子构成炮管  
- 方向为直线（横 / 竖）  

---

### 发射机制

设炮管长度为 $n$，等级为 $k$：

- 前 $n-2$ 格受到影响  
- 影响方式：
  - 空格视为 0  
  - 敌方棋子视为负值  
  - 增加 $k$ 等级  
- 奇数距离额外 +1  

---

### 吃子

当目标棋子满足：

- 至少一半相邻格为己方棋子  
- 且周围总等级优势成立  

则可以被吃掉，并变为己方 1级棋子
